La Memoria RAM

Actualmente, la RAM más utilizada es la DDR4, y es la que más se presenta en equipos como computadoras y teléfonos móviles. Es sumamente importante, ya que si tiene poca capacidad o es lenta, retrasará las funciones de la CPU y del chip gráfico.

DDR5

Definición

Se podría decir que la RAM siglas para Random Access Memory o Memoria de Acceso Aleatorio es una tabla que contiene la dirección donde está cierto dato y el contenido de este.

Funcionamiento

La memoria dispone de un registro de direcciones y un registro de datos. En el registro de direcciones se almacena la dirección en la que se almacenará o leerá un dato, y en el registro de datos se almacena ese dato leído o que se almacenará.

Un poco de historia...

Uno de los primeros tipos de memoria RAM fue la memoria de núcleo magnético, desarrollada entre 1949 y 1952 y usada en muchos computadores hasta el desarrollo de circuitos integrados a finales de los años 60 y principios de los 70.

Tipos de Memoria

DDR SDRAM (Double Data Rate Synchronous Dynamic Random-Access Memory, SDRAM de tasa de datos doble)
DDR2 SDRAM (Double Data Rate type two SDRAM, SDRAM de tasa de datos doble de tipo dos)
DDR3 SDRAM (Double Data Rate type three SDRAM, SDRAM de tasa de datos doble de tipo tres)
DDR4 SDRAM (Double Data Rate type four SDRAM, SDRAM de tasa de datos doble de tipo cuatro).
DDR5 SDRAM (Double Data Rate type five SDRAM, SDRAM de tasa de datos doble de tipo cinco).
